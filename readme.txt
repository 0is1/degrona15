== Description ==
De Gröna 15 theme is clean, modern, election-focused, and designed for election candidates based on FoundationPress theme.

== Installation ==

1. Download https://dl.dropboxusercontent.com/u/11862254/vihreat/FoundationPress.zip, https://dl.dropboxusercontent.com/u/11862254/vihreat/degrona15.zip and https://dl.dropboxusercontent.com/u/11862254/vihreat/degrona-ehdokas.zip to your computer.
2. In your admin panel, go to Appearance -> Themes and click the Add New button.
3. Click Upload and Choose File, then select the FoundationPress.zip file. Click Install Now.
4. Click Return to Themes page.
5. Click Upload and Choose File, then select the degrona15.zip file. Click Install Now.
6. Then go to Plugins -> Add New and click the Upload Plugin button.
7. Choose File, then select the degrona_ehdokas.zip file. Click Install Now.
8. Click Activate Plugin.
9. Click Return to Plugin Installer and type "Simple Image Widget" to Search Plugins input.
10. Click Install Simple Image Widget and Activate Plugin.
11. Then go back to Appearance -> Themes and click Activate on the De Gröna 15 theme.

== Technical details ==

* child theme, based on [FoundationPress theme](https://github.com/olefredrik/foundationpress)
